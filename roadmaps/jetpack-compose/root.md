# Jetpack Compose

Jetpack Compose est le toolkit UI moderne et déclaratif d'Android, développé par Google. Il remplace le système de vues XML traditionnel par des fonctions Kotlin annotées `@Composable`, permettant de décrire l'interface en fonction de l'état.

```kotlin
@Composable
fun Greeting(name: String) {
    Text(text = "Bonjour, $name!")
}
```
