# Rust

Rust est un langage de programmation système qui garantit la sécurité
mémoire **sans garbage collector**, grâce à son système d'ownership.

## Philosophie

- **Sécurité** — pas de null pointer, pas de dangling reference
- **Performance** — zéro coût d'abstraction
- **Concurrence** — "fearless concurrency" garanti par le compilateur
