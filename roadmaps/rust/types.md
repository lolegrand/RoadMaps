# Système de types

Rust est **statiquement et fortement typé** avec inférence de types.

## Primitifs

| Type | Exemples |
|------|----------|
| Entiers | `i8` `i32` `i64` `u8` `u32` `usize` |
| Flottants | `f32` `f64` |
| Booléen | `bool` |
| Caractère | `char` (Unicode 4 octets) |
| Chaîne | `&str` (emprunt) / `String` (owned) |
| Tableau fixe | `[T; N]` |
| Vecteur | `Vec<T>` |
| Tuple | `(T, U, V)` |
