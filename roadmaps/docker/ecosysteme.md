# Écosystème

Au-delà de Docker lui-même, l'écosystème des outils de conteneurisation est riche : alternatives à Docker, orchestrateurs, outils de développement local et services managés.

## Paysage des outils

```
Outils de build      : Docker BuildKit, Podman, Buildah, Kaniko
Runtimes             : containerd, CRI-O, runc
Orchestration locale : Docker Compose, Podman Compose
Orchestration prod   : Kubernetes, Docker Swarm, Nomad
Registries           : Docker Hub, GHCR, ECR, GCR, Harbor
Dev local            : DevContainers, Tilt, Skaffold
Sécurité             : Trivy, Grype, Falco, Docker Scout
```
