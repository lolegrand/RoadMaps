# Débogage

Déboguer efficacement fait la différence entre 5 minutes et 5 heures pour résoudre un bug. Les outils modernes — DevTools, debugger Node.js, source maps — permettent de naviguer dans le code en temps réel.

## Principe : "shift left" sur les erreurs

```
console.log(?) → Debugger (breakpoints) → Tests automatisés → Types (TypeScript)
     Lent             Rapide               Permanent          Compilation
```
