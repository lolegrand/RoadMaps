# Déploiement

Déployer une application web consiste à la rendre accessible sur internet. Les plateformes modernes automatisent le CI/CD, le CDN et les certificats TLS.

## Choisir la bonne plateforme

| Plateforme | Type | Points forts | Cas d'usage |
|-----------|------|-------------|-------------|
| **Vercel** | PaaS | DX, Edge, Next.js | Frontend, Fullstack JS |
| **Netlify** | PaaS | CI/CD, Forms, Edge | Sites statiques, JAMstack |
| **Railway** | PaaS | Bases de données incluses | Node.js, fullstack |
| **Render** | PaaS | Docker natif, simple | APIs, workers |
| **Fly.io** | Containers | Multi-région, Docker | Apps avec état |
| **AWS / GCP / Azure** | IaaS | Contrôle total | Grandes entreprises |
| **Cloudflare Pages** | Edge | CDN mondial, Workers | Sites statiques + APIs edge |
