# Outillage

L'outillage moderne transforme l'expérience de développement : gestion des dépendances avec npm/pnpm, bundling avec Vite, qualité avec ESLint + Prettier.

## Les outils incontournables

| Outil | Catégorie | Rôle |
|-------|-----------|------|
| `npm` / `pnpm` | Package manager | Gérer les dépendances |
| `Vite` | Bundler/Dev server | Build rapide, HMR |
| `ESLint` | Linter | Détecter les erreurs de code |
| `Prettier` | Formateur | Style de code uniforme |
| `TypeScript` | Transpileur | Typage statique |
| `Vitest` | Test runner | Tests unitaires rapides |
